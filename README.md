# User Dashboard Next

Современная панель управления пользователями, построенная на базе **Next.js 14** с использованием App Router.

## 🚀 Технологический стек

* **Framework:** [Next.js 14](https://nextjs.org) (React 18)
* **State Management:** [Zustand](https://github.com)
* **Styles:** [SASS (SCSS)](https://sass-lang.com)
* **Icons:** [React Icons](https://github.io)
* **Type Safety:** [TypeScript](https://typescriptlang.org)

## 🧠 Почему выбраны именно эти технологии?

Проект спроектирован с упором на производительность, масштабируемость и простоту поддержки:

1. **Next.js 14 (App Router):** Использование серверных компонентов (RSC) позволяет минимизировать объем JavaScript на
   клиенте и ускорить первую загрузку страниц.
2. **Zustand вместо Redux:** Выбран за минималистичный API и отсутствие лишнего шаблонного кода (boilerplate). Это
   обеспечивает быстрое управление состоянием без усложнения архитектуры.
3. **SASS & CSS Modules:** Для стилизации выбраны SCSS модули. Это дает мощь препроцессоров (переменные, миксины,
   вложенность), сохраняя при этом строгую изоляцию стилей компонентов, что исключает конфликты имен.
4. **TypeScript:** Обеспечивает безопасность разработки, предотвращая типичные ошибки на этапе написания кода и улучшая
   автодополнение в IDE.
5. **Prettier & ESLint:** Настроены для поддержания единого стиля кода в команде и автоматического исправления мелких
   ошибок форматирования.

## 🛠 Скрипты для разработки

| Команда | Описание |
| :--- | :--- |
| `npm run dev` | Запуск сервера разработки на `localhost:3000`. |
| `npm run build` | Сборка проекта для продакшена. |
| `npm run start` | Запуск собранного приложения. |
| `npm run lint` | Проверка кода линтером. |
| `npm run type-check` | Проверка типов TypeScript. |
| `npm run format` | Форматирование кода через Prettier. |

## 📦 Установка и запуск

1. **Клонируй репозиторий:**
   ```bash
   git clone <url_вашего_репозитория>
   cd user-dashboard-next
   ```
 
2. **Установи зависимости:**
   ```bash
   npm install
   ```

3. **Запусти проект:**
   ```bash
   npm run dev
   ```

## 📂 Структура проекта

* `/app` — маршрутизация и страницы (App Router).
* `/components` — переиспользуемые UI компоненты.
* `/store` — конфигурация хранилищ Zustand.
* `/styles` — глобальные стили и миксины (Sass).
